# Meu-primeiro-projeto-
Durante esta atividade aprendi os conceitos básicos de controle de versão, Git e GitHub. Entendi que o controle de versão é importante para salvar as alterações de um projeto e evitar a perda de arquivos. O Git é o programa que faz esse controle no computador, e o GitHub é o site onde podemos guardar e compartilhar esses projetos com outras pessoas.

Criar minha conta e o repositório foi mais fácil do que eu esperava. Aprendi a adicionar arquivos, editar o README e entender como os commits registram as mudanças. Um dos desafios foi entender a diferença entre Git e GitHub, mas depois percebi que o Git funciona localmente e o GitHub é a parte online, onde tudo fica salvo.

Essa atividade me mostrou como o Git e o GitHub são essenciais para quem trabalha com tecnologia, programação ou projetos colaborativos. Agora sei como criar e organizar repositórios e pretendo continuar praticando para usar essas ferramentas com mais confiança no futuro.
